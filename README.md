# yzpLongclick
基于angularjs的长按指令
##EXAMPLE CODE
```html
<!DOCTYPE html>
<html>
	<head>
		<meta charset="utf-8">
		<link rel="stylesheet" type="text/css" href="style.css">
	</head>
	<body ng-app="app">
		<button yzp-longclick="doSome()" >mybutton</button>
		<script type="text/javascript" src="angular.js"></script>
		<script type="text/javascript" src="yzpLongclick.js"></script>
	</body>
</html>
```
